# 连连看游戏部署指南

## 方法一：使用 Netlify Drop（最简单，推荐）

1. 打开浏览器，访问：https://app.netlify.com/drop
2. 将整个文件夹拖到网页上的拖放区域
3. 等待几秒钟，就会自动生成一个链接
4. 把链接分享给朋友即可！

## 方法二：使用 Tiiny.host

1. 访问：https://tiiny.host/
2. 点击 "Upload" 或 "Start for free"
3. 选择 index.html 文件上传
4. 获得一个可分享的链接（免费版链接有效期较短）

## 方法三：使用 Surge.sh（需要安装 Node.js）

1. 安装 Node.js 后，运行：npm install -g surge
2. 在项目文件夹运行：surge
3. 按照提示操作，会生成一个链接

## 方法四：使用 GitHub Pages

1. 在 GitHub 创建新仓库
2. 上传 index.html 文件
3. 在仓库设置中启用 GitHub Pages
4. 获得链接：https://你的用户名.github.io/仓库名/

---

**推荐使用 Netlify Drop，最简单快捷！**
