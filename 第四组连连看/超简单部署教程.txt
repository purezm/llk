🌟 方法一：Netlify Drop（最推荐！30秒搞定）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第1步：打开网站
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在浏览器地址栏输入：
https://app.netlify.com/drop

按回车键

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第2步：注册账号
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 点击右上角的 "Sign up" 按钮
2. 填写邮箱和密码
3. 点击 "Create account"
4. 检查邮箱，点击验证邮件里的链接

💡 提示：也可以用 GitHub 或 Google 账号登录

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第3步：上传文件夹
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开"我的电脑"或"文件资源管理器"
2. 找到这个文件夹：d:\桌面\代码\第四组连连看
3. 用鼠标选中整个文件夹
4. 按住鼠标左键，拖到浏览器网页的虚线框里
5. 松开鼠标

💡 提示：虚线框上写着 "Drag and drop your site output folder here"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第4步：等待部署
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 你会看到进度条在加载
2. 等待大约 10-30 秒
3. 看到 "Site is live!" 就表示成功了！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第5步：复制链接
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 页面上会显示一个链接，比如：https://xxxxx.netlify.app
2. 点击链接旁边的复制按钮（📋 图标）
3. 把链接粘贴到微信、QQ 发给朋友

✅ 完成！你的朋友现在可以点击链接玩游戏了！


🌟 方法二：Tiiny.host（上传单个文件）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第1步：打开网站
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在浏览器地址栏输入：
https://tiiny.host/

按回车键

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第2步：上传文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 点击页面中间的 "Upload" 按钮
2. 选择 index.html 文件（在 d:\桌面\代码\第四组连连看 文件夹里）
3. 输入网站名称（随便填，比如 "连连看游戏"）
4. 点击 "Upload" 按钮

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第3步：获取链接
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 等待几秒钟
2. 页面上会显示你的网站链接
3. 复制链接发给朋友

⚠️ 注意：免费版链接只能用 7-30 天


🌟 方法三：GitHub Pages（永久免费）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第1步：注册 GitHub
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开 https://github.com
2. 点击右上角的 "Sign up"
3. 填写用户名、邮箱、密码
4. 点击 "Create account"
5. 验证邮箱

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第2步：创建仓库
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 登录后，点击右上角的 "+" 号
2. 选择 "New repository"
3. 仓库名称填写：lianliankan-game
4. 勾选 "Public"（公开）
5. 点击 "Create repository"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第3步：上传文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 在新创建的仓库页面，点击 "uploading an existing file"
2. 点击 "choose your files"
3. 选择 index.html 文件
4. 在下方输入：添加连连看游戏
5. 点击 "Commit changes" 按钮

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第4步：开启 GitHub Pages
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 点击仓库顶部的 "Settings"（设置）
2. 在左侧菜单找到 "Pages"
3. 在 "Branch" 选项选择 "main"
4. 点击 "Save" 按钮

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第5步：获取链接
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 等待 1-2 分钟
2. 刷新页面
3. 在顶部会显示你的网站链接
4. 格式是：https://你的用户名.github.io/lianliankan-game/

❓ 常见问题

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 我看不懂英文怎么办？
A: 右键点击网页，选择"翻译成中文"

Q: 上传后链接打不开？
A: 等待 1-2 分钟，有时需要时间部署

Q: 链接有效期多久？
A: Netlify 和 GitHub Pages 是永久的
   Tiiny.host 免费版只有 7-30 天

Q: 可以修改游戏吗？
A: 可以！修改 index.html 文件后重新上传即可


🎯 最快操作流程（Netlify Drop）

1. 打开 https://app.netlify.com/drop
2. 注册/登录
3. 拖放 d:\桌面\代码\第四组连连看 文件夹到网页
4. 等待 30 秒
5. 复制链接发给朋友
