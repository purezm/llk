# 连连看游戏部署教程（超详细版）

## 方法一：Netlify Drop（最简单，推荐！）

### 第一步：打开网站
1. 打开你的浏览器（Chrome、Edge、Firefox 都可以）
2. 在地址栏输入：`https://app.netlify.com/drop`
3. 按回车键

### 第二步：注册/登录
1. 如果看到登录页面，点击右上角的 "Sign up"（注册）
2. 可以选择用邮箱注册，或者用 GitHub、Google 账号登录
3. 填写邮箱和密码，点击 "Create account"
4. 检查邮箱，点击验证链接

### 第三步：上传文件
1. 登录后，你会看到一个大的虚线框，上面写着 "Drag and drop your site output folder here"
2. 打开你的电脑文件管理器
3. 找到这个文件夹：`d:\桌面\代码\第四组连连看`
4. 用鼠标选中整个文件夹
5. 按住鼠标左键，把文件夹拖到浏览器网页的虚线框里
6. 松开鼠标

### 第四步：等待部署
1. 你会看到进度条在加载
2. 等待大约 10-30 秒
3. 看到 "Site is live!" 就表示成功了！

### 第五步：获取链接
1. 页面上会显示一个链接，比如：`https://xxxxx.netlify.app`
2. 点击链接旁边的复制按钮（📋图标）
3. 把链接粘贴到微信、QQ 等地方发给朋友

## 方法二：Tiiny.host（上传单个文件）

### 第一步：打开网站
1. 打开浏览器
2. 在地址栏输入：`https://tiiny.host/`
3. 按回车键

### 第二步：上传文件
1. 点击页面中间的 "Upload" 按钮
2. 或者点击 "Start for free"
3. 会弹出一个文件选择窗口
4. 找到并选择 `index.html` 文件（在 `d:\桌面\代码\第四组连连看` 文件夹里）
5. 点击"打开"

### 第三步：设置网站信息
1. 输入网站名称（随便填，比如 "连连看游戏"）
2. 选择免费套餐（Free）
3. 点击 "Upload" 按钮

### 第四步：获取链接
1. 等待几秒钟
2. 页面上会显示你的网站链接
3. 复制链接发给朋友

⚠️ **注意**：免费版链接通常只能用 7-30 天

---

## 方法三：GitHub Pages（永久免费，但步骤较多）

### 第一步：注册 GitHub
1. 打开 https://github.com
2. 点击右上角的 "Sign up"
3. 填写用户名、邮箱、密码
4. 点击 "Create account"
5. 验证邮箱

### 第二步：创建仓库
1. 登录后，点击右上角的 "+" 号
2. 选择 "New repository"
3. 仓库名称填写：`lianliankan-game`
4. 勾选 "Public"（公开）
5. 点击 "Create repository"

### 第三步：上传文件
1. 在新创建的仓库页面，点击 "uploading an existing file"
2. 点击 "choose your files"
3. 选择 `index.html` 文件
4. 在下方 "Commit changes" 输入：`添加连连看游戏`
5. 点击 "Commit changes" 按钮

### 第四步：开启 GitHub Pages
1. 点击仓库顶部的 "Settings"（设置）
2. 在左侧菜单找到 "Pages"
3. 在 "Build and deployment" 下面的 "Branch" 选项
4. 选择 "main" 分支
5. 点击 "Save" 按钮

### 第五步：获取链接
1. 等待 1-2 分钟
2. 刷新页面
3. 在顶部会显示你的网站链接
4. 格式是：`https://你的用户名.github.io/lianliankan-game/`

---

## 💡 推荐选择

- **如果你想要最快**：用 Netlify Drop（方法一）
- **如果你想要永久链接**：用 GitHub Pages（方法三）
- **如果你只是临时分享**：用 Tiiny.host（方法二）

## ❓ 常见问题

**Q: 我看不懂英文怎么办？**
A: 可以使用浏览器翻译功能，右键点击页面选择"翻译成中文"

**Q: 上传后链接打不开？**
A: 等待 1-2 分钟，有时需要时间部署

**Q: 链接有效期多久？**
A: Netlify 和 GitHub Pages 是永久的，Tiiny.host 免费版只有 7-30 天

**Q: 可以修改游戏吗？**
A: 可以！修改 index.html 文件后重新上传即可
